# **Hospital-Management-System- SQL Database Design**





This project implements a relational database schema for a Hospital Management System using SQL DDL statements.



\## Features

###### 

**Creation of core entities:**



&#x20; \* Employee

&#x20; \* Department

&#x20; \* Project

&#x20; \* Dependent



**Implementation of:**



&#x20; \* Primary Keys

&#x20; \* Foreign Keys

&#x20; \* Entity Relationships

&#x20; \* Many-to-Many relationships using associative tables

&#x20; \* Constraints and referential integrity rules



**## Database Concepts Applied**



\* Relational Mapping

\* SQL DDL (CREATE TABLE)

\* Relationship Constraints

\* Department Management

\* Employee Supervision

\* Employee–Project Assignment with Working Hours



**## Objective**



The purpose of this project is to convert an ER model into a fully implemented relational database schema using SQL.
